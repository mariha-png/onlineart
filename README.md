# 🎨 Online Art Gallery

A modern and responsive **Online Art Gallery** web application developed using **ASP.NET MVC**, **C#**, **SQL Server**, **Entity Framework**, and **Bootstrap**.

## 📖 About

Online Art Gallery is a web application that enables users to explore artworks, view artwork details, manage a shopping cart, and complete purchases through a responsive and user-friendly interface.

---

## ✨ Features

- 🎨 Browse Art Collection
- 🖼️ Artwork Details
- 🔍 Search Artworks
- 🛒 Shopping Cart
- ❤️ Wishlist
- 👤 User Authentication
- 💳 Checkout System
- 📞 Contact Page
- 📱 Responsive Design

2. Restore NuGet packages.
3. Configure the SQL Server connection string.
4. Build and run the project.
